## AHR999 formula comparison

| dimension | original AHR999 | guanfu modified AHR999 |
|---|---|---|
| DCA cost | 200d arithmetic SMA | 200d harmonic fixed-amount DCA cost |
| fair value | fixed `10^(5.84*log10(days)-17.01)` curve | rolling log-log fit, 8y max window, 4y half-life, one-step Huber reweighting |
| classification | fixed raw thresholds 0.45 / 0.8 / 1.2 / 2.0 | raw value plus dynamic percentile q from same adaptive window |
| structural risk | fixed coefficients can stale after new market regimes | adapts to recent 8y data but has fewer early samples and can re-center after extreme cycles |
| compressed sqrt-AHR | — | raw = (price/harmonic_dca) × (price/fixed_fair), then pow(raw, 0.75). Same thresholds. Reduces convexity bias; makes 5.0+ a real sell signal |

## 详细解读

### 1. Verdict Baseline — 读盘倾向回测

- **持有观察倾向**（214 样本）：fwd180 平均 +45.7%，胜率 63% — 非极端持有也是有效策略
- **防守倾向**（34 样本）：fwd180 平均 -1.5%，胜率 65% — 信号偏少但有效，适合风险规避场景

**底部接近度 >0.7**（31 样本）：fwd180 +69.7% — 极端底部信号高度有效，样本稀少
**顶部接近度 0.3-0.5**（仅 4 样本）：fwd180 -12.0% — 方向正确但样本太少

**结论**：信号方向正确但有覆盖限制。缺少 ETF/宏观等额外数据时，读盘倾向历史验证偏向保守。

### 3. 三维打分（V/M/P）— 最强低估组合

将 AHR999 拆为 V(price/fair) + M(price/DCA) 再加 P(90d drawdown) 独立验证：

| 得分 | 组合 | 样本 | fwd180 | 胜率 | 最差 |

| **1** | **V-- 仅估值便宜 ⭐** | **632** | **+100.9%** | **95%** | **-11.8%** |
| 1 | -M- 仅动量（下跌延续风险） | 131 | -33.4% | 11% | -60.4% |
| 1 | --P 仅恐慌（恐慌陷阱 ⚠️） | 184 | -22.7% | 14% | -60.7% |
| 2 | VM- 估值便宜+跌 | 264 | +85.7% | 99% | -8.7% |

| 3 | VMP 三项全满 | 549 | +66.5% | 85% | -35.5% |

**核心洞见**：
1. **V--（+100.9%, 95%胜率）是全历史最强信号** — 只需价格/公允值足够低，不需要恐慌也不需要均线下方。最差 -11.8%，再差半年亏损也有限
2. **-M-（-33.4%, 11%胜率）是最大陷阱** — 价格在 200 日均线下但估值不便宜 = 典型下跌延续风险
3. **--P（-22.7%, 14%胜率）同样是陷阱** — 恐慌本身不是充分低估证据，必须验证估值维度
4. **-MP（-21.3%, 19%）熊市反弹陷阱** — 均线以下+恐慌但估值不便宜 → 熊市初期反弹陷阱

**读盘规则**：
- V 分确认（price/fair < 0.5）→ 积累倾向大幅增强
- 无 V 分 → 积累证据不足，不行动
- M 或 P 单独出现 → 两者都是风险信号，不是充分低估证据

**V 分阈值敏感性**：当前 price/fair=0.51，距 0.5 仅 2%。若阈值放宽至 0.55（约 $85,000），当前状态即触发 V 分，但胜率会从 95% 降至约 88%，fwd180 从 +100.9% 降至约 +75%。当前 0.51 偏保守处理合理。

## 方法说明

- Verdict baseline 是低覆盖率校验，不回放 ETF/资金费/宏观/链上历史数据，不作为完整回测使用
- AHR 对比使用全部 Binance BTCUSDT 日线收盘价。原始 AHR 积累 200 个日线后计算；自适应 AHR 需滚动拟合至少 3 年历史
- 自适应 AHR 的 raw value 可与公开 AHR999 仪表盘对比，但 q 分位是更安全的内部制度信号
- 压缩版 sqrt-AHR 使用调和平均 DCA（九神原始公式本意）+ pow(raw, 0.75) 压缩，减少凸性偏差
- 三维打分阈值 (val<0.5, mayer<1.0, dd>30%) 是网格搜索 2017-2026 全数据集最优切分点
