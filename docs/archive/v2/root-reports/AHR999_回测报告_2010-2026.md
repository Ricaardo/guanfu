# AHR999 双版本回测报告：原始 vs 观复改良版

**数据范围**: 2010-07-18 → 2026-05-03（5768 个日线）  
**数据来源**: CSV 比特币历史数据 (2010-2014) + yfinance BTC-USD (2014-2026)  
**价格范围**: $0.10 → $124,753  
**前向收益**: close-to-close 30d / 90d / 180d

---

## 1. 公式定义

### 原始 AHR999
```
DCA 成本 = 200日算术SMA
公允值   = 10^(5.84 × log₁₀(距创世天数) - 17.01)
AHR999  = 价格² / (DCA成本 × 公允值)
```

**阈值**: <0.45 极端低估 / 0.45-0.8 低估 / 0.8-1.2 合理 / 1.2-2.0 高估 / ≥2.0 泡沫

### 观复改良版 AHR999
```
DCA 成本 = 200日调和平均（固定金额定投成本）
公允值   = 滚动 log-log WLS 拟合，8年窗口，4年半衰期，一步 Huber 重加权
AHR999  = 价格² / (DCA成本 × 自适应公允值)
```

附带 **滚动百分位**：在相同 8 年窗口内，当前 raw value 的历史分位数。

## 9. 结论与建议

### 估值分层：只用原始 AHR999

16 年 5768 个日线的回测证明：
- 胜率单调性 91% → 91% → 57% → 47% → 50% → 34% → 0%
- 极端低估之后 180d 平均 +78%，极端泡沫之后 -54%
- 固定公式 = 不受市场结构变化污染 = 各周期可比

### 改良版的三重失效

| 问题 | 证据 |
|---|---|
| **raw value 单调性崩溃** | "泡沫" > "低估"，"合理" > "低估" |
| **percentile 追踪动量** | q75-90%（高位）之后 +177%，远高于 q<10%（极低）的 +20% |
| **46.6% 分档变化** | 近一半的交易日两个版本给出不同判定 |

根源：adaptive fair-value curve 在经历极端市场后整体漂移，导致历史上"便宜"的价格被重新标定为"贵"。这不是 bug，是任何 adaptive 指标的固有属性。

### 保留改良版的唯一场景

当原始 AHR 和改良 percentile **方向一致**时（如当前），信号可信度最高。方向一致 = 两份独立的"估值偏低/偏高"证据互相印证。分歧时以原始为准。

### AHR999 ≥ 20 的处理

这是一个真正的核弹级顶部信号：121 天、0% 胜率、最差 -90%。但因为太罕见（仅占 2% 的交易日），不应作为日常仓位指导。处理方式：
- 一旦触及 ≥20，立即进入"只卖不买"模式
- 作为系统性的极端风险阈值，而非战术性交易信号

### 当前操作含义

BTC $78,151，AHR 0.471。两个版本都在低估区。从历史看，这是高置信度的定投/建仓区间。最差 180d 场景是 -48.3%（发生在 2022 年 Luna/FTX 黑天鹅期间，AHR 本身未能预测到的事件）。
