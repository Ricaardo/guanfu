## 2. 全网最佳实践调研

### 2.1 原始作者的真实公式（ahr999.com）

原始作者定义的 DCA 成本是「固定金额每日定投的平均成本」：

```
DCA = 200 / Σ(1/P_i)    // 调和平均，不是算术 SMA
```

**大部分第三方实现（CoinGlass、非小号等）用 SMA 替代了调和平均。** 两者差异：SMA 被高价日拉高，调和平均被低价日拉低。调和平均始终 ≤ SMA，差值在高波动期可达 3-5%。

### 2.2 Power Law 模型的最新演进

| 来源 | 方法 | 关键发现 |
|---|---|---|
| **Santostasi** (Bitcoin Power Law) | log-log OLS 回归，Price = A × Days^5.7 | R² ≈ 0.95，幂律关系稳健 |
| **Perrenod** (2025) | AR(2)-GLS 修正自相关 | 斜率从 5.68 陡化至 5.89，DW 统计量从 0.03 修复至 2.01 |
| **PlanC / Quantile Model** | 分位数回归取代 OLS | 给概率分布而非点估计 |
| **Porkopolis** (2025.01 更新) | **用时点系数取代全历史系数** | "不再用全历史回归线做分位带，改用当时时点的系数" |

Porkopolis 的改进最关键：它在每个历史时点用「到当时为止的数据」拟合回归线，然后计算价格偏离该时点回归线的倍数，再做百分位。这避免了对未来数据的污染——**这与观复改良版的思路一致，但用分位数回归带而非连续自适应曲线，避免了过度拟合。**

### 2.3 AHR999 阈值分布（ahr999 原始博文）

| 区间 | 时间占比 | 策略 |
|---|---|---|
| <0.45 | 8.5% | 抄底 |

| 1.2-5.0 | 29.3% | 等待起飞 |

注意：原始作者自己把 >5（不是 >2）定义为泡沫区，且明确表态"不准备卖币所以不考虑"。

**Divergence = Danger.** Consensus = noise during trends.

## 4. 关键洞察：为什么"一致"失效而"分歧"有效

```
牛市进行中：
  价格 ↗ → 原始AHR ↗ (变贵) → 改良公允值 ↗ (追价格) → 改良 percentile ↗
  → 两个版本都说"贵" → 但牛市继续 → fwd180 为正
  
熊市进行中：  
  价格 ↘ → 原始AHR ↘ (变便宜) → 改良公允值 ↘ (滞后) → 改良 percentile ↗
  → 原始说"便宜"但百分位说"偏贵" → 分歧 → 熊市继续 → fwd180 为负
```

**分歧发生在市场转向期**——当价格已从高位回落但改良版公允值还没跟上来（滞后），百分位会反常偏高。这时原始 AHR 可能已经回到"合理"甚至"低估"，但百分位还在喊贵。这种分歧**不是噪声，是转向信号。**

## 5. 最终推荐方案

### 5.1 核心指标：Harmonic AHR999 + Sqrt 压缩

```
DCA 成本    = 200 / Σ(1/P_i)              // 调和平均（原始作者公式）
公允值      = 10^(5.84 × log₁₀(days) - 17.01)  // 固定幂律（不变）
AHR999_opt = price^1.5 / (DCA^0.75 × fair^0.75)  // 降低凸性
```

| 阈值 | 含义 | 策略 |
|---|---|---|
| <0.45 | 极端低估 | 重仓 DCA / 抄底 |
| 0.45-0.8 | 低估 | 标准定投 |
| 0.8-1.2 | 合理 | 维持仓位 |
| 1.2-2.0 | 偏高 | 暂停定投，持有 |
| 2.0-5.0 | 高估 | 开始减仓 |
| 5.0-20.0 | **泡沫** | **大幅减仓** ← 之前这是假阳性，现已修复 |
| ≥20.0 | **极端泡沫** | **清仓** ← 0% 胜率 |

### 5.2 辅助确认：Divergence Detector

当原始 AHR 与改良百分位**方向相反**时，是市场转向预警：

```
if ahr_orig < 0.8 AND ahr_mod_q > 55:
    → "原始说便宜，但自适应说目前仍偏贵 — 可能还在下跌中继，分批建仓不要一次全仓"
    
if ahr_orig > 1.2 AND ahr_mod_q < 35:
    → "原始说贵，但自适应说这轮周期不算高 — 危险！可能是熊市初期反弹"
    
if ahr_orig > 2.0 AND ahr_mod_q < 50:
    → "原始泡沫 + 自适应不极端 — 最危险组合，历史 -34%~-53%"
```

### 5.3 放弃的方案

| 方案 | 原因 |
|---|---|
| 自适应公允值曲线 | 橡皮筋效应破坏单调性 |
| 改良 percentile 作为独立指标 | 跟踪动量非均值回归 |
| Relative AHR (vs 200d median) | 最差的单调性 |
| AHR Momentum 作为反转信号 | 趋势跟踪，非反转 |
| Ensemble "双低估"/"双高估"一致信号 | 在趋势市中完全无效 |

### 5.4 保留观复改良版的唯一价值

**Divergence detector.** 当固定公式和自适应公式打架时，不是该信谁的问题——打架本身就是信号。
