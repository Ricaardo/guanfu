# 观复 · 宣传文案

## 长版（公众号 / 博客）

```
致虚极，守静笃。万物并作，吾以观复。

——《道德经》第十六章

观复，是一个 BTC 盘面。

它不画线，不喊单，不输出评分。
只做一件事：把 8 个维度同时呈现，每个指标标注它在历史中的分位。

判断是你的。工具只是工具。

开源 MIT，本地运行：github.com/Ricaardo/guanfu
```
