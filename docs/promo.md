# 观复 · 宣传文案

## 长版（公众号 / 博客）

```
致虚极，守静笃。万物并作，吾以观复。

——《道德经》第十六章

观复。BTC 盘面。

矿工在扩张，还是在投降？
机构在买，还是在卖？
现在的价格，在历史上同一个周期里，是贵了还是便宜了？

它不给出答案——它帮你看到更多维度。
看到周期往复，看到贪婪与恐惧来来去去。
然后你自己做判断。

这就是「观」的全部含义。
它不是预测未来。它只是帮你把现在看得更清楚。

盘面在此。判断是你的。

开源 MIT，本地运行：github.com/Ricaardo/guanfu
```
