# 观复 · 宣传文案

## 长版（公众号 / 博客）

```
致虚极，守静笃。万物并作，吾以观复。

——《道德经》第十六章

观复，是一个 BTC 盘面。

但它不画线，不喊单。它做的事只有一件：
让 8 个维度的信号同时呈现，不替代你的判断。

「观复」的意思很简单：
你把画面拉远一点，就能看出哪些是潮汐，哪些是浪花。

矿工在扩张，还是在投降？
机构在买，还是在卖？
现在的价格，在历史上同一个减半周期里，是贵了还是便宜了？

它不给出答案——它帮你看到更多维度。
看到周期往复，看到人性的贪婪与恐惧来来去去，
然后你自己做判断。

这就是「观」的全部含义。

它不是预测未来。它只是帮你把现在看得更清楚。

开源，MIT，本地运行。
代码不经过服务器，数据不经过中间人。
这是你的盘面，不是别人的。

github.com/Ricaardo/guanfu
```
