# 观复 · 宣传文案

## 长版（公众号 / 博客）

```
致虚极，守静笃。万物并作，吾以观复。

——《道德经》第十六章

观复，是一个 BTC 盘面。但不是你以为的那种盘面。
它不画线，不喊单，不输出"65分建议观望"。

它做的事只有一件：让 8 个维度的 44 个指标同时呈现，
每个指标标注它在历史中的分位——现在贵，还是便宜？
矿工在扩张，还是在投降？机构在买，还是在卖？

「观复」的意思是：万物纷纭起作，我守在静处看它们的往复回归。
你不必每日盯盘。你只需要在关键的时刻，看一眼这些指标在历史上意味着什么。

当前盘面可做历史相似度复盘：
把今天的 8 域 q 分位，与归档历史盘面逐项比较，找出最接近的时刻。

不是预测未来。是告诉你：历史上信号长这样的时候，后面发生过什么。
相似度和回测口径由 `guanfu-similar` 与 `docs/backtest-methodology.md` 生成，不手写挑样本。
判断是你自己的。工具只是帮你观。

44 个指标自己拉、自己算、自己存历史。Go 单二进制，本地运行，数据不经过服务器。
开源 MIT：github.com/Ricaardo/guanfu
```
